Vorabveröffentlichung DEMIS-FHIR-Profile

Auf dieser Seite werden kommende Profilanpassungen der jeweiligen DEMIS-Pakete vorab veröffentlich. Die Vorab-Versionen befinden sich in der alpha- bzw. rc-Entwicklungsstufe und bieten einen ersten Einblick in die kommenden Inhalte. Die Simplifier-Seiten der FHIR-Profile beinhalten weiterhin den Profil-Stand, welcher aktuell vom DEMIS-Backend unterstützt wird.

FHIR-Profile Basismeldeinhalte (rki.demis.common)

Das Paket rki.demis.common beinhaltet eine Reihe an DEMIS-Basis-Ressourcen wie z.B. betroffene Person oder Melder. Spezifische DEMIS-Pakete wie rki.demis.laboratory oder rki.demis.disease binden dieses als Dependency ein.

Aktuell gibt es keine Vorabveröffentlichungen für das rki.demis.common-Paket. Seit 19.02.2025 wird die Version rki.demis.common-2.0.0 durch das DEMIS-Backend unterstützt. Das entsprechende Simplifier-Projekt https://simplifier.net/rki.demis.common wurde aktualisiert.

FHIR-Profile Erregernachweismeldung (rki.demis.laboratory)

Das Paket rki.demis.laboratory beinhaltet die Ressourcen für die Erregernachweismeldung.

Aktuell gibt es keine Vorabveröffentlichungen für das rki.demis.laboratory-Paket. Seit 19.02.2025 wird die Version rki.demis.laboratory-3.0.0 durch das DEMIS-Backend unterstützt. Das entsprechende Simplifier-Projekt https://simplifier.net/rki.demis.laboratory wurde aktualisiert.

FHIR-Profile Integrierte Genomische Surveillance (rki.demis.igs)

Das Paket rki.demis.igs erweitert die Ressourcen des rki.demis.laboratory-Paketes für die Integrierte Genomische Surveillance (IGS).

Aktuell gibt es keine Vorabveröffentlichungen für das rki.demis.igs-Paket. Seit 11.03.2025 wird die Version rki.demis.igs-3.0.0 durch das DEMIS-Backend unterstützt. Das entsprechende Simplifier-Projekt https://simplifier.net/rki.demis.igs wurde aktualisiert.

FHIR-Profile Erkrankungsmeldung (rki.demis.disease)

Das Paket rki.demis.disease beinhaltet die Ressourcen für die Erkrankungsmeldung.

Aktuell gibt es keine Vorabveröffentlichungen für das rki.demis.disease-Paket. Seit 03.02.2025 wird die Version rki.demis.disease-2.0.2 durch das DEMIS-Backend unterstützt. Das entsprechende Simplifier-Projekt https://simplifier.net/rki.demis.disease wurde aktualisiert.
