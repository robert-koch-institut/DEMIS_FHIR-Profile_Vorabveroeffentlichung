Vorabveröffentlichung DEMIS-FHIR-Profile

Auf dieser Seite werden kommende Profilanpassungen der jeweiligen DEMIS-Pakete vorab veröffentlich. Die Vorab-Versionen befinden sich in der rc-Entwicklungsstufe und bieten einen ersten Einblick in die kommenden Inhalte. Die Simplifier-Seiten der FHIR-Profile beinhalten weiterhin den Profil-Stand, welcher aktuell vom DEMIS-Backend unterstützt wird.

FHIR-Profile Basismeldeinhalte (rki.demis.common)

Das Paket rki.demis.common beinhaltet eine Reihe an DEMIS-Basis-Ressourcen wie z.B. betroffene Person oder Melder. Spezifische DEMIS-Pakete wie rki.demis.laboratory oder rki.demis.disease binden dieses als Dependency ein.

    • Extension und ValueSet County entfernt

    • ConceptMaps IHEHealthcareFacilityCodes2DEMISOrganizationTypeCodes, IHEOutsideHealthcareFacilityCodes2DEMISOrganizationTypeCodes und ISIKOrganizationTypeCodes2DEMISOrganizationTypeCodes hinzugefügt

FHIR-Profile Integrierte Genomische Surveillance (rki.demis.igs)

Das Paket rki.demis.igs erweitert die Ressourcen des rki.demis.laboratory-Paketes für die Integrierte Genomische Surveillance (IGS).

Aktuell gibt es keine Vorabveröffentlichungen für das rki.demis.igs-Paket. Seit 11.03.2025 wird die Version rki.demis.igs-3.0.0 durch das DEMIS-Backend unterstützt. Das entsprechende Simplifier-Projekt https://simplifier.net/rki.demis.igs wurde aktualisiert.

FHIR-Profile Erkrankungsmeldung (rki.demis.disease)

Das Paket rki.demis.disease beinhaltet die Ressourcen für die Erkrankungsmeldung.

Aktuell gibt es keine Vorabveröffentlichungen für das rki.demis.disease-Paket. Seit 09.04.2025 wird die Version rki.demis.disease-2.2.0 durch das DEMIS-Backend unterstützt. Das entsprechende Simplifier-Projekt https://simplifier.net/rki.demis.disease wurde aktualisiert.
