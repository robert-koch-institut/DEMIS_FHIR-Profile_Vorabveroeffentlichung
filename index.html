Vorabveröffentlichung DEMIS-FHIR-Profile

Auf dieser Seite werden kommende Profilanpassungen der jeweiligen DEMIS-Pakete vorab veröffentlich. Die Vorab-Versionen befinden sich in der rc-Entwicklungsstufe und bieten einen ersten Einblick in die kommenden Inhalte. Die Simplifier-Seiten der FHIR-Profile beinhalten weiterhin den Profil-Stand, welcher aktuell von der DEMIS-Produktivumgebung unterstützt wird.

FHIR-Profile Basismeldeinhalte (rki.demis.common)

Das Paket rki.demis.common beinhaltet eine Reihe an DEMIS-Basis-Ressourcen wie z.B. betroffene Person oder Melder. Spezifische DEMIS-Pakete wie rki.demis.laboratory oder rki.demis.disease binden dieses als Dependency ein.

Seit 10.09.2025 wird die Version rki.demis.common-2.2.1 in der DEMIS-Produktivumgebung unterstützt. Das entsprechende Simplifier-Projekt https://simplifier.net/rki.demis.common wurde aktualisiert.
Seit 21.10.2025 wird die Version rki.demis.common-3.0.0-rc.4 in der DEMIS-Testumgebung und seit 18.11.2025 in der DEMIS-Produktivumgebung unterstützt. Das entsprechende Simplifier-Projekt https://simplifier.net/rki.demis.common.strict wurde aktualisiert.

FHIR-Profile Erregernachweismeldung (rki.demis.laboratory)

Das Paket rki.demis.laboratory beinhaltet die Ressourcen für die Erregernachweismeldung.

Seit 10.09.2025 wird die Version rki.demis.laboratory-3.4.0 in der DEMIS-Produktivumgebung unterstützt. Das entsprechende Simplifier-Projekt https://simplifier.net/rki.demis.laboratory wurde aktualisiert.
Seit 21.10.2025 wird die Version rki.demis.laboratory-5.0.0-rc.1 in der DEMIS-Testumgebung und seit 18.11.2025 in der DEMIS-Produktivumgebung unterstützt. In dieser Version sind die Ressourcen für den Meldungstyp "Meldungen gemäß IfSG §7 Absatz 3" hinzugefügt wurden. Das entsprechende Simplifier-Projekt https://simplifier.net/rki.demis.laboratory.strict wurde aktualisiert.

FHIR-Profile Integrierte Genomische Surveillance (rki.demis.igs)

Das Paket rki.demis.igs erweitert die Ressourcen des rki.demis.laboratory-Paketes für die Integrierte Genomische Surveillance (IGS).

Aktuell gibt es keine Vorabveröffentlichungen für das rki.demis.igs-Paket. Seit 11.03.2025 wird die Version rki.demis.igs-3.0.0 in der DEMIS-Produktivumgebung unterstützt. Das entsprechende Simplifier-Projekt https://simplifier.net/rki.demis.igs wurde aktualisiert.

FHIR-Profile Erkrankungsmeldung (rki.demis.disease)

Das Paket rki.demis.disease beinhaltet die Ressourcen für die Erkrankungsmeldung.

Seit 10.09.2025 wird die Version rki.demis.disease-2.2.2 in der DEMIS-Produktivumgebung unterstützt. Das entsprechende Simplifier-Projekt https://simplifier.net/rki.demis.disease wurde aktualisiert.
Seit 21.10.2025 wird die Version rki.demis.disease-3.0.0-rc.1 in der DEMIS-Testumgebung unterstützt. In dieser Version sind die Ressourcen für den Meldungstyp "Meldungen gemäß IfSG §7 Absatz 3" hinzugefügt wurden.
